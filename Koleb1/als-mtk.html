Matematika

Produk Pertama

X = Satu produk Mug ekoprint
y = Satu Produk Tempat sampah
z = Satu produk Lampu

Model Matematika

x + y + z = 125.000.....1
x + 2y + z = 185.000.....2
2x + y + 3z = 155.000.....3

Penyelesaian

  • Eliminasi => y pers 1 dan 2

    • x + y + z = 125.000 /×2/ 2x + 2y + 2z = 250.000
    x + 2y + z = 185.000 /×1/ x + 2y + z = 185.000-
    x + z = 65.000...4

  • Eliminasi y => 1 dan 3

    • x + y + z = 125.000
    2x + y + z = 155.000-
    -x = -30.000...50

  • Eliminasi pers 4 dan 5

    • x + z = 65.000
    -x = -30.000+
    z = 35.000

  • Substitusi z = 35.000 ke pers 4
    • x + 35.000 = 65.000
    x = 65.000 - 35.000
    x = 30.000

  • Substitusi z = 35.000, x = 30.000 ke pers 1
    • x + y + z = 125.000
    30.000 + y + 35.000 = 125.000
    y = 125.000 - 30.000 - 35.000
    y = 60.000

{( HP : 30.000 , 35.000 , 60.000 )}

Produk Kedua

x = satu produk lampu
y = satu produk mug
z = satu produk bunga

Model Matematika
1. x + y + z = 115.000
2. x + 2y + z = 145.000
3. x + 2y + 2z = 170.000

Penyelesaian:

Eliminasi x dari pers 2 dan 1
x + 2y + z = 145.000
x + y + z = 115.000-
y = 30.000

Eliminasi x ke pers 3 dan 1
x + 2y + 2z = 170.000
x + y + z = 115.000-
y + z = 55.000...4

Substitusi y = 30.000 ke persamaan 4
30.000 + z = 55.000
z = 55.000 - 30.000
z = 25.000

Subtitusi y = 30.000, z = 25.000 ke pers 1
x + 30.000 + 25.000 = 115.000
x = 115.000 - 30.000 - 25.000
x = 60.000

{(HP : 60.000, 30.000, 25.000)}
